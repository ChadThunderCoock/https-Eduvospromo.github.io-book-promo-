The Brick Protocol is a small set of conversational habits designed to reduce narrative inflation when interacting with large language models. It was born from the observation that sycophantic, mythologising language is a default output mode — not a feature request.

It is not anti-creativity. It asks only that grandiosity be earned, not auto-suggested.
